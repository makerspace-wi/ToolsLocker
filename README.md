# ToolsLocker
Schrank mit 10 Türen für wertvolle Werkzeuge - Entnahme per Mitglieder RFID-Chip

Brain Storming:
- Schrank mit 10 Fächern
- Verriegelung mechanisch (mit Hand zudrücken)
- Entriegelung elektromagnetisch (12V DC 2A Puls)
